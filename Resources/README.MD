## 📚 Resources

The `Resources/` folder contains the official project handout and supporting materials used to build this GRC graduation project.

### Main Resource File

- 📄 **GRC Graduation Project Handout Round 4.pdf**

This file represents the official project requirement document for the DEPI GRC Round 4 graduation project. It defines the project title, overview, required deliverables, evaluation criteria, and expected learning outcomes.

The handout requires the team to prepare a feasibility study and IT governance framework for a German car manufacturer expanding into Egypt. The project must cover business feasibility, governance structure, compliance requirements, ISO 27001 extension, policies and procedures, technical controls, audit readiness, and final presentation deliverables.

### Resource Purpose

The resources folder is used to keep all reference materials that support:

- 📊 Business Feasibility Study
- 🏛️ IT Governance Framework
- ⚠️ Risk Register and Risk Treatment Plan
- 🔐 ISO 27001 Extension Plan
- ⚖️ GDPR and Egyptian PDPL Compliance
- 📚 Policies, Procedures, and Standards
- 🔎 Audit and Monitoring Readiness
- 🎤 Final Presentation Preparation
